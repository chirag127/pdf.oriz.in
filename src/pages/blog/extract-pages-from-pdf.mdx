### Single Page Extraction

Select any single page from your PDF to create a new one-page document. This is useful for:
- Extracting signature pages
- Pulling cover pages
- Saving specific illustrations or data tables
- Creating single-page handouts

### Page Range Extraction

Extract continuous page ranges:
- First five pages (1-5)
- Last ten pages (20-30)
- Middle section (10-15)

### Multiple Selection Extraction

Combine multiple non-contiguous selections:
- Pages 1-3 then skip 4 then pages 5-7

- Alternating specific pages

## Handling Large Documents

### Multi-Page PDFs

Extracting from large PDFs (50+ pages) requires careful planning:

1. Open the original PDF and identify page ranges
2. Note exact start and end pages for each section
3. Plan extractions in logical order
4. Extract each section and rename for clarity

### Performance Considerations

For very large documents:
- Processing takes slightly longer
- Preview generation may take a moment
- Extraction itself remains fast
- Download size depends on selected pages only

## Troubleshooting Common Issues

### Incorrect Page Numbers

<TipBox title="Page Numbering" type="warning">
PDFs use zero-based internal page numbering (first page is page 0 or 1), but our tool displays visually accurate page numbers that match what you see in your PDF reader.
</TipBox>

If extracted pages don't match your expectations:
- Verify the original document page numbering
- Check if the PDF has cover pages counted
- Preview the original again to confirm page selection

### Extraction Fails

Common reasons extraction might fail:
- Invalid page range specified (pages don't exist)
- File is corrupted or protected
- Network interruption during upload

Solutions:
- Double-check page range validity
- Verify file opens normally in PDF reader
- Re-upload and try again

### File Size Increases

Sometimes extracted files appear larger due to how PDF compression works internally. Our tool optimizes each extraction, but some variations occur based on content type.
