## Understanding PDF Password Types

Before attempting to unlock a PDF, it's important to understand what type of protection you're dealing with. This helps determine the best approach.

### User Password (Open Password)

A user password is required to even open the PDF document. Without this password, the file cannot be viewed at all. Without the password, you'll see:
- A prompt requesting the password
- No content visible without entering the correct password
- The entire document encrypted

### Owner Password (Permissions Password)

An owner password controls permissions like printing, copying text, and editing. Documents with only owner password protection typically:
- Open without requiring a password
- Display all content normally
- But show "secret" or restricted icons when trying to print, copy, or edit
- Block certain actions without the owner password

<TipBox title="Important" type="warning">
You should only unlock PDFs you legally own or have proper authorization to access. Removing password protection from documents you don't own or aren't authorized to open may be illegal.
</TipBox>
